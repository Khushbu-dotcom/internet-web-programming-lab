# Internet & Web Programming Lab 

This repository contains all programs and lab work performed during the **Internet & Web Programming Lab** course.
It includes implementations using **HTML, CSS, JavaScript, PHP, MySQL, and AJAX**, covering both client-side and server-side concepts.

##  How to Run

###  HTML & JavaScript Programs

* Open `.html` files in any web browser

###  Node.js Programs

1. Install Node.js
2. Open terminal
3. Run:

```
node filename.js
```

###  PHP Programs

1. Install XAMPP
2. Start **Apache** and **MySQL**
3. Place project folder in:

```
C:\xampp\htdocs\
```

4. Open browser:

```
http://localhost/internet-web-programming-lab/
```
